About Our Project

CNNs in Our Project

This basic CNN architecture includes an input layer for receiving data, a convolution layer for feature extraction, a max-pooling layer for down-sampling, and a softmax layer for classification. The model is initialized with random weights and biases, and during training, these parameters are adjusted to minimize the difference between the predicted and actual class labels..

RISC-V Processor on FPGA

Implementing a RISC-V processor on an FPGA involves designing a digital circuit that executes RISC-V instructions. The flexibility of FPGAs allows for customization and adaptation to various applications, making them a suitable platform for prototyping and development.

CNN on FPGA

Implementing a CNN on an FPGA provides advantages such as parallel processing, custom hardware acceleration, low latency, flexibility, energy efficiency, optimized memory access, efficient resource utilization, suitability for real-time processing, and high throughput. These factors make FPGA-based CNN implementations well-suited for a range of applications, from edge devices to data centers, where performance and efficiency are crucial.
